文件读取

文件读取可以读取文本文件

记得在auth里面加上 file

首先我们在插件的目录创建一个1.txt文件

插件的内容如下

// 获取API接口
const router =xBlog.router
const file = xBlog.file
// 注册路由
router.registerRouter("GET","",function(context){
    file.read("1.txt",function (err,res){
        if (!err){
            router.response.ResponseOk(context,res)
        }
    })
})
