内容板块

# get_header

获取头部信息，使用的时候，需要指明一下当前是哪个界面，方便对不同界面显示不同的头部信息

<?php get_header('post'); ?>
1

# get_footer

获取尾部信息

# xy_posts

显示文章列表，常用于首页文章列表显示

# xy_left_side

加载左边侧边栏

# xy_right_side

加载右边侧边栏

# xy_access

显示注册登录界面

# xy_diary

显示日记列表

# xy_comments

显示文章评论，需要传入文章id

<?php xy_comments(get_post_id()); ?>
